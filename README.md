# Hilla RPG
Hilla RPG é uma versão mobile do [RPG](https://github.com/chelo84/rpg), um jogo de RPG de texto.
Hilla RPG foi desenvolvido em Java com a ferramenta de desenvolvimento mobile chamada Codename One, com arquivos de configuração dos elementos do jogo em json.

## Como instalar
* Instale o jogo no seu celular (Android) através do arquivo .apk que se encontra na pasta `apk` do repositório
